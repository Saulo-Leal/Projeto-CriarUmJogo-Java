# Projeto-CriarUmJogo-Java
Objetivo e motivação do trabalho
Desenvolver um jogo com a linguagem Java que utiliza os conceitos de Linguagem de Programação Orientada a Objetos. Um jogo em Java oferece uma série de vantagens didáticas e práticas. Aqui estão algumas delas:
Aprendizado de Programação Orientada a Objetos (POO): Java é uma linguagem de programação orientada a objetos popular e poderosa. Desenvolver um jogo em Java permite que os aprendizes pratiquem os conceitos fundamentais de POO, como classes, objetos, encapsulamento, herança e polimorfismo.
Entendimento de Estruturas de Dados: Jogos exigem frequentemente o uso eficiente de estruturas de dados para armazenar e manipular informações do jogo, como mapas, personagens, itens, etc. Desenvolver um jogo em Java oferece a oportunidade de explorar diferentes estruturas de dados e entender como elas afetam o desempenho e a jogabilidade.
